Contributions to `digital-garden` are welcome! There's always an open issue with the current tracking list, which contains the proposed features for the next release you can get your hands on:
- Open an issue for non-trivial changes; small fixes can be submitted as a PR.
- Follow the project Principles: see `README.md#principles`.
- Keep changes isolated, add tests where relevant, and include a clear PR description.

Maintainers will review and respond promptly. Thanks!

## Setting up a development environment

Recommended editor is [VSCode](https://code.visualstudio.com/) with full support for JavaScript/TypeScript and Nu shell scripts.

### Extensions for VSCode

- [vscode-dbml](https://marketplace.visualstudio.com/items?itemName=matt-meyers.vscode-dbml): DBML syntax highlighting and preview.
- [DBML Entity-Relationship Diagrams visualizer](https://marketplace.visualstudio.com/items?itemName=bocovo.dbml-erd-visualizer): visualize DBML files as ER diagrams.
- [vscode-nushell-lang](https://marketplace.visualstudio.com/items?itemName=TheNuProjectContributors.vscode-nushell-lang): Nushell syntax highlighting.
